Retail Pro 8 Exporter Download

by Swarm-Mobile

Connect your Retail Pro POS data to your Swarm account. By installing the exporter you'll be able to continuously track the transactions and revenue of your store.

Installation

  • Download the SwarmRetailPro8Exporter-1.7.2.zip and extract all files to a folder of your choice.
  • Start the Swarm8Installer.msi from the folder it has been extracted to.
  • When prompted for your Swarm ID enter the token Swarm has provided for you
  • Fill in your path to your Rpro folder. It should look like C:/RetailPro/RPro or "your drive":/"your install folder"/RPro, it is visible in the status bar of your Retail Pro client
  • Enter your workstation id, visible in the status bar of your Retail Pro client
  • Your past POS data will be also uploaded to Swarm, you can select the start date during the installation, this value should not be more than 4 months old
  • Windows might asks you to allow the installer to make changes to your computer, in such case click "Yes"
  • Should you have any problems please read the detailed installation guide
If your receive a "Source file not found" error during installation you've probably forgotten to extract the contents of the archive before launching the installer. Please right click on the downloaded file, click "Extract All..." and try again.

System requirements

  • Microsoft Windows XP SP3 or later
  • Microsoft .NET 4.0 or later
  • Retail Pro 8 (known to work best with 8.60)
  • Internet connectivity
  • Administrative privileges for Windows
